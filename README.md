# Controle de Tarefas

## Sobre o Projeto
O projeto Controle de Tarefas tem como objetivo auxiliar na organização de atividades do dia a dia. Ele permite cadastrar, visualizar e marcar tarefas como concluídas, facilitando o acompanhamento de compromissos pessoais, acadêmicos ou profissionais de forma simples.

**Projeto:** Controle de Tarefas  
**Problema que resolve:** Dificuldade em organizar e acompanhar tarefas do dia a dia.
